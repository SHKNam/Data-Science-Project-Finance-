# PRD: DART 금융 데이터 분석 프로젝트

## 목표
DART(전자공시시스템) API를 활용하여 한국 상장기업의 재무 데이터를 수집, 분석, 시각화한다.

## 사용자
- 금융 데이터 분석에 관심 있는 데이터 사이언티스트 / 개인 투자자

## 핵심 기능
1. DART API를 통한 기업 재무제표 데이터 수집
2. 데이터 전처리 및 탐색적 데이터 분석 (EDA)
3. 재무 지표 기반 시각화 및 인사이트 도출

## MVP 제외 사항
- 실시간 주가 연동
- 웹 대시보드 / UI
- 자동 매매 시스템

## 데이터 소스
- DART OpenAPI (https://opendart.fss.or.kr/)
- 일일 API 호출 제한 있음 (무료)
